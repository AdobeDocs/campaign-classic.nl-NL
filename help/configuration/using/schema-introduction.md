# Schema-referentie {#schema-reference}

Wanneer het uitgeven van een schema, is een goedkeuringssysteem dat op het bronschema (xtk :srcSchema wordt gebaseerd) beschikbaar. Sommige fouten kunnen ook worden ontdekt wanneer het bijwerken van het gegevensbestand gebruikend de &quot;update van de structuur van het Gegevensbestand...&quot;medewerker.

Standaard zijn in Adobe Campaign-schema&#39;s alle Booleaanse tekstkenmerken &quot;false&quot;. Om hen te activeren, moet u de attributen in het schema specificeren en zijn waarde plaatsen aan &quot;waar&quot;.
